## 4. Data Preprocessing Pipeline

**Instruction:**
Construct a preprocessing pipeline to ensure data is model-ready.

1.  **Feature Selection:**
    *   Retain the specified features: `size`, `weight`, `sweetness`, `softness`, `harvest_time`, `ripeness`, `acidity`.
    *   Drop the `quality` column from the feature matrix `X`.

2.  **Scaling:**

    *   Fit the scaler on the training split (see Step 5).

3.  **Splitting:**
    *   Split the data into Training (70%) and Testing (30%) sets using `train_test_split`.
    *   Set `stratify=y` to maintain class distribution.
    *   Record the split indices for reproducibility.

## 5. Model Selection and Training (Critical Constraint)

**Instruction:**
Select and train models that satisfy the **5-minute maximum training time** constraint. Use state-of-the-art tabular learning algorithms optimized for speed.

1.  **Model Architecture:**
    *   **Primary Model:** `LightGBM Classifier`. It offers state-of-the-art performance on tabular data with fast training speeds compared to XGBoost or Neural Networks.
    *   **Secondary Model (Fallback):** `RandomForest Classifier` (if LightGBM fails to converge within time).
    *   **Do Not Use:** Deep Neural Networks (too slow for 5-minute constraint) or Grid Search (too computationally expensive).

2.  **Hyperparameter Optimization (HPO):**
    *   Use **Optuna** for hyperparameter search, configured as follows:
        *   **Max Trials:** 20 (Strictly limit to ensure time compliance).
        *   **Time Budget:** Set a hard stop if the trial exceeds 3 minutes total.
        *   **Study Direction:** `maximize` (Targeting F1-Score).
    *   **Search Space:**
        *   `num_leaves`: [15, 31, 63, 127]
        *   `learning_rate`: [0.01, 0.05, 0.1]
        *   `n_estimators`: [50, 100, 200]
        *   `feature_fraction`: [0.6, 0.8, 1.0]
        *   `class_weight`: `balanced` (Enable automatically).

3.  **Execution:**
    *   Train the model on the training split.
    *   Monitor execution time using a system timer.
    *   **Hard Stop:** If cumulative training time reaches 4 minutes and 30 seconds, stop the HPO immediately and save the best model found so far.
